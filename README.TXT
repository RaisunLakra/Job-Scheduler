# jOB ScheDuLer

Developed a Job Seduler which takes Job title, Job Description, Job priority (1 -> highest, 10 -> Lowest), Arrive time, completion time(approx. time)
and return real time remainder for completion time based on the sorting algo (shortest Job First, FCFS, Priority algo, etc).

Input : Job Title
        Job Desc
        Job Priority
        Arrival Time
        Completion Time

Output : Table of Content

        Schedular -> arrival time, priorit, burst time, completion time, average waiting time, jobId
